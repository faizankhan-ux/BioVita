# 🏥 BioVita — AI Powered Emergency Patient Identification System

> **Saving Lives in the Golden Hour**

BioVita is a modern healthcare web application designed to **instantly identify unconscious patients** and provide doctors with **critical medical information in seconds** using biometric and QR-based access.

---

# 🚀 Problem

In emergency situations, especially road accidents:

- Doctors **cannot identify unconscious patients**
- No access to:
  - Blood group  
  - Allergies  
  - Medications  
- **15–30 minutes are wasted** in testing  
- This delay can **cost lives during the Golden Hour**

---

# 💡 Solution

BioVita provides a **digital emergency identity system**:

- Register once  
- Store critical medical data securely  
- Instantly retrieve patient data using:
  - QR Code  
  - Face Recognition (demo)  
  - Fingerprint (simulated)  

👉 **No phone, no card, no patient interaction required**

---

# 🧠 MVP Features

## 👤 Citizen Side
- Multi-step registration form  
- Stores:
  - Blood group  
  - Allergies  
  - Medications  
  - Surgeries  
  - Emergency contacts  
- Upload face image  
- Generates unique QR code  

---

## 🩺 Doctor Side
- Login dashboard  
- Emergency search using:
  - QR scan  
  - Face scan (demo)  
- Instant patient data display  

---

## 📊 Dashboard
- Patient health card view  
- Critical info highlighted:
  - Blood group (RED)  
  - Allergies (warning)  
- Emergency contact actions
